## Getting Started

To get a local copy up and running follow these simple example steps.


### Prerequisites
- Download Visual Studio Code.
- Download Git.

### Install

1. Clone the repo
```sh
   git clone https://github.com/odaialazzeh/protrfolio.git
```
2. Install NPM packages
```sh
   npm install
```
